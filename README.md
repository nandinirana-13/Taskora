# Taskora - Smart Productivity Web App

Taskora is a modern web-based productivity application designed to help users focus, organize, and achieve more efficiently. It integrates three essential tools into one unified interface.

## Features

### Focus Sprint
A timer based on the Pomodoro technique that helps users maintain focus by working in structured intervals with short breaks.

### Mission Planner
A task management system that allows users to:
- Add tasks (missions)
- Delete completed tasks
- Track daily productivity

### Idea Vault
A notes system that enables users to:
- Create and edit notes
- Save ideas using browser storage
- Organize important information

## How to Run

1. Download or clone the project  
2. Open the folder in Visual Studio Code  
3. Right-click on `index.html`  
4. Select "Open with Live Server"  

## Usage

- Focus Sprint: Start the timer and work in focused intervals  
- Mission Planner: Add and manage daily tasks  
- Idea Vault: Store and organize notes
